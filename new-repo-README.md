# Client Portal

🏢 A complete portal management system for enterprise use.

## ✨ Features

- 📊 **Dashboard** - System overview and statistics
- 👥 **Account Management** - User account CRUD operations  
- 🔐 **Role Management** - Role and permission configuration
- 📋 **Menu Management** - System menu structure management
- 📝 **Audit Logs** - Operation tracking and logging
