# 自定义表头功能使用示例

## 功能说明

Demo现在支持通过对话直接指定表头来生成对应的表格，系统会自动识别字段类型并生成合适的示例数据。

## 支持的格式

### 基本格式
```
表头：字段1，字段2，字段3
```

### 其他支持的格式
```
表头:字段1,字段2,字段3
创建表格，字段：字段1，字段2，字段3
生成表格，包含：字段1，字段2，字段3
表格字段：字段1，字段2，字段3
列名：字段1，字段2，字段3
```

## 使用示例

### 1. 员工管理表格
**输入：** `表头：姓名，年龄，部门，职位，入职时间`

**效果：**
- 自动生成员工管理表格
- 姓名字段显示中文姓名
- 年龄字段显示20-60的随机数字
- 部门字段显示各种部门名称
- 职位字段显示不同职位
- 入职时间显示日期格式

### 2. 商品管理表格
**输入：** `表头：商品名称，品牌，价格，库存，销量`

**效果：**
- 商品名称显示各种产品名
- 品牌显示知名品牌
- 价格显示货币格式（¥xxx.xx）
- 库存和销量显示数字，带不同颜色

### 3. 学生成绩表格
**输入：** `表头：学号，姓名，班级，语文，数学，英语`

**效果：**
- 学号显示编号格式
- 姓名显示学生姓名
- 班级显示班级信息
- 各科成绩显示分数

### 4. 订单管理表格
**输入：** `表头：订单号，客户名称，金额，状态，创建时间`

**效果：**
- 订单号显示编号格式
- 客户名称显示客户信息
- 金额显示货币格式
- 状态显示不同状态标签（激活/禁用/待审核）
- 创建时间显示日期

## 智能数据生成规则

系统会根据字段名称智能生成对应类型的数据：

### 姓名类字段
- **触发词：** 姓名、名称、用户名
- **生成数据：** 中文姓名、用户名等

### 时间类字段
- **触发词：** 时间、日期、创建时间、更新时间
- **生成数据：** 日期格式数据

### 金额类字段
- **触发词：** 价格、金额、费用
- **生成数据：** 货币格式（¥xxx.xx）

### 数量类字段
- **触发词：** 数量、库存、销量
- **生成数据：** 数字，带颜色标识

### 状态类字段
- **触发词：** 状态
- **生成数据：** 状态标签（激活/禁用/待审核）

### 联系方式字段
- **触发词：** 电话、手机、邮箱
- **生成数据：** 电话号码、邮箱地址

### ID类字段
- **触发词：** ID、id、编号、学号、订单号
- **生成数据：** 编号格式，等宽字体显示

### 地址类字段
- **触发词：** 地址、城市
- **生成数据：** 地址信息

## 高级功能

### 1. 字段样式自动适配
- 不同类型的字段会自动应用不同的显示样式
- 金额字段显示红色
- 数量字段显示绿色
- 时间字段显示灰色小字
- ID字段使用等宽字体

### 2. 状态标签
- 状态字段自动生成彩色标签
- 激活状态：绿色背景
- 禁用状态：红色背景
- 待审核状态：黄色背景

### 3. 操作按钮
- 自动添加编辑、删除按钮
- 支持新增、搜索、导出等操作

## 使用技巧

1. **字段命名要准确**：使用常见的字段名称可以获得更好的数据生成效果
2. **合理的字段数量**：建议3-8个字段，太多会影响显示效果
3. **字段顺序**：重要字段放在前面，操作相关字段放在后面
4. **中文支持**：完全支持中文字段名，效果更好
