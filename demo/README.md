# 对话式页面设计 Demo

这是一个展示对话式页面设计功能的演示版本，通过简化的界面和交互来展示核心功能。

## 功能特性

### 🤖 智能对话
- 自然语言理解用户需求
- 上下文记忆和多轮对话
- 智能建议和快速操作
- **自定义表头识别**：支持"表头：xxx，xxx，xxx"格式

### 🎨 页面生成
- 基于模板自动生成页面
- 实时预览页面效果
- 支持多种页面类型
- **智能数据生成**：根据字段名自动生成合适的示例数据

### 💾 页面管理
- 保存页面到本地存储
- 查看页面配置信息
- 导出页面代码

## 使用方法

### 1. 打开Demo
直接在浏览器中打开 `index.html` 文件即可开始使用。

### 2. 开始对话
在左侧对话框中输入您的需求，例如：

#### 预设页面类型：
- "我想创建一个用户管理页面"
- "帮我做一个角色管理页面"
- "需要一个菜单管理界面"

#### 自定义表格（新功能）：
- "表头：姓名，年龄，部门，职位"
- "表头：产品名称，价格，库存，销量"
- "表头：学号，姓名，班级，成绩"
- "创建表格，字段：订单号，客户名称，金额，状态"

系统会自动识别表头信息并生成对应的表格结构和示例数据。

### 3. 查看预览
系统会在右侧实时生成页面预览，您可以：
- 查看页面效果
- 查看页面配置
- 保存页面到本地

### 4. 保存页面
点击"保存页面"按钮，输入页面名称和描述即可保存。
