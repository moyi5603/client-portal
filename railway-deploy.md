# Railway 部署链接

## 直接部署链接

点击下面的链接直接部署到 Railway：

**方法 1：使用模板链接**
https://railway.app/template/nodejs?referralCode=bonus&repo=https://github.com/moyi5603/client-portal

**方法 2：直接导入**
https://railway.app/new?template=https://github.com/moyi5603/client-portal

**方法 3：手动输入仓库 URL**
1. 访问：https://railway.app/new
2. 选择 "Deploy from GitHub repo"
3. 在搜索框中输入：`moyi5603/client-portal`
4. 或者直接粘贴：`https://github.com/moyi5603/client-portal`

## 如果仍然找不到仓库

这可能是因为：
1. 仓库是私有的（需要设为公开）
2. GitHub 权限问题
3. Railway 缓存延迟

### 解决方案：
1. 确保 GitHub 仓库是公开的
2. 在 Railway 中重新授权 GitHub
3. 等待几分钟后重试
